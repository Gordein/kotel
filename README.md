# Котёл

Сплиттер расходов на квартиру для троих (Сэм, Люда, Микита), валюта **zł**.
Flask + HTMX + Alpine + SQLite, ставится как PWA. Один экран, спокойный дизайн.

## Что умеет
- **Один экран:** сверху — запись новой траты, ниже — твой баланс и лента по месяцам.
- Вход по **PIN** (Сэм 111, Люда 222, Микита 333). PWA запоминает телефон.
- Трата = что **ты** заплатил, делим на выбранных. Живая подсказка «X zł на каждого».
- Необязательное **примечание** к трате — в ленте разворачивается по ссылке.
- **Аренда** — обычная трата категории «Квартира» (каждый вносит свою часть).
- Баланс показывает только твои долги; «Закрыть долг» — когда есть что закрывать. Никакого графа и лишних цифр.
- **Удалить** можно свою запись. Лента сгруппирована по месяцам, со временем (Варшава) и итогом за месяц.
- **Живое обновление:** баланс/лента подтягиваются каждые ~15с, новые записи помечаются точкой.
- Без потери данных: неизменяемые записи, баланс на чтении, оптимистичная блокировка, idempotency, SQLite WAL.

## Запуск (Windows — cmd или PowerShell)

    cd C:\Users\latas\Downloads\kotel
    python -m venv .venv
    .venv\Scripts\python.exe -m pip install -r requirements.txt
    set SECRET_KEY=change-me
    .venv\Scripts\python.exe -m flask --app wsgi init-db
    .venv\Scripts\python.exe -m flask --app wsgi run --host 0.0.0.0 --port 8000

PowerShell: ключ задаётся как `$env:SECRET_KEY="change-me"`. Остановить сервер — `Ctrl+C`
в его окне. При старте печатаются два адреса (`127.0.0.1` и `192.168.x.x`) — это один
сервер на всех интерфейсах; второй адрес для телефонов в той же Wi-Fi.

Сбросить данные / логины: останови сервер, удали `kotel.db*`, снова `init-db`.
