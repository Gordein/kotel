# Котёл

Сплиттер расходов на квартиру для троих (Сэм, Люда, Микита), валюта **zł**.
Flask + HTMX + Alpine + SQLite, устанавливается как PWA. Дизайн в стиле Claude.

## Что умеет
- Вход **только по PIN** (Сэм 111, Люда 222, Микита 333 — меняются в профиле). PWA запоминает телефон.
- Траты с делёжкой: **кто заплатил** и **на кого делим** (любое подмножество) — видно сразу.
- Шаблон «Аренда» в один тап (наём+коммуналка одной категорией «Квартира»).
- **Эго-центричный баланс:** «ты должен / тебе должны», граф только твоих долгов, минимальные переводы.
- Закрытие долгов (нал/перевод) с подстановкой суммы, общая лента, комментарии.
- Защита от потери данных: неизменяемые записи, баланс на чтении, оптимистичная блокировка, idempotency, SQLite WAL.

## Запуск (Windows — cmd или PowerShell)

    python -m venv .venv
    .venv\Scripts\python.exe -m pip install -r requirements.txt
    set SECRET_KEY=change-me
    .venv\Scripts\python.exe -m flask --app wsgi init-db
    .venv\Scripts\python.exe -m flask --app wsgi run --host 0.0.0.0 --port 8000

В PowerShell задавай ключ так: `$env:SECRET_KEY="change-me"`.

Открой `http://127.0.0.1:8000` на компе или `http://<IP-компа>:8000` с телефона в той же
Wi-Fi. При старте печатаются оба адреса — это **один** сервер на всех интерфейсах (`--host 0.0.0.0`),
а не два разных. «Добавить на экран» → установится как приложение. Войди по PIN, смени его в «Профиле».

Сбросить данные / перевыпустить логины: останови сервер (Ctrl+C), удали `kotel.db*`, снова `init-db`.
